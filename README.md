## 🚀 Proyectos Destacados

### 🏆 **Atlas** - Sistema de Gestión de Proyectos Formativos
> **Proyecto Estrella | SENA Software Factory**

<details>
<summary><strong>Ver detalles del proyecto</strong></summary>

**🎯 Descripción:** Ecosistema de microservicios para gestión educativa que permite a instituciones manejar proyectos formativos con metodologías ágiles.

**🏗️ Arquitectura:**
```
┌─────────────────┐    ┌──────────────────┐    ┌─────────────────┐
│   Frontend      │◄──►│   API Gateway    │◄──►│  Microservicios │
│   Next.js 14    │    │   Spring Boot    │    │   - Usuarios    │
│   + TypeScript  │    │   + GraphQL      │    │   - Proyectos   │
└─────────────────┘    │   Federation     │    │   - Académico   │
                       └──────────────────┘    └─────────────────┘
```

**🛠️ Stack Tecnológico:**
- **Backend:** Java 17 + Spring Boot 3 + Netflix DGS + MongoDB
- **Frontend:** Next.js 14 + TypeScript + Tailwind CSS + Zustand
- **Arquitectura:** Microservices + GraphQL Federation

**✨ Características Clave:**
- 🔗 **GraphQL Federation** para unificación de múltiples microservicios
- 📊 **Dashboard en tiempo real** con métricas de proyectos
- 🏢 **Multi-tenant** para diferentes instituciones
- 🔄 **Metodologías ágiles** (Scrum, Kanban)

**📈 Resultados:**
- ⚡ Mejora del 70% en tiempos de gestión académica
- 👥 Soporte para 500+ usuarios simultáneos
- 📱 Interfaz responsive con 95% de satisfacción

</details>
